First, as with all communications from a debt collector, you should keep the letter in a safe place. Second, if you would like, we will take a look at the letter and give you our opinion, for which there is no charge.

Please scan the letter, including anything on the back of any pages and, if possible, the envelope as well. Please save it as a PDF file and email it to us. Also, if it is the first letter, it probably contains a notice about a 30-day period for you to dispute the debt and request information about the original creditor. If you choose to exercise any of those rights, please see below regarding disputing a debt.
